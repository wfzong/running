# 从零开始，构建一个完整的前端项目
这不是一个教程，更多的算是一个记录，记录如何从零开始，构建一个完整的前端项目，主要会涉及到以下技术或者知识点：
- 域名注册
- centos 7 服务器的管理
- nginx 配置
- Jenkins的安装配置，自动化部署php项目、vue项目、vue SSR项目
- php、mysql的配置，开发一些简单的api接口 
- 开发项目的vue版本
- 开发项目的vue ssr版本
- 开发项目的微信小程序版本
- 开发项目的weex版本

下面会对以上项目分别介绍
